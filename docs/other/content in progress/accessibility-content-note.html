Content and Accessibility Note

Accessibility

Hopefully it's clear that Braille Toolbox project is all about creating more access to braille. This means to people who are interested in it, or would benefit from knowing it better. Thus, we've strived to make this 100% accessible via modern web practices. We target compliance with the Web Content Accessibility Guidelines (WCAG) version 2.1, targeting AA for now and AAA before long. Do not hesitate to report anything on the site that doesn't comply with that intent with this link. Notable exception to this are new complex features that are being worked on. Any such page is marked with with a "under construction" tag. Feel free to report problems with those.

Content Imperfection

This site is an iterative project. Certain functions, such as Write in Braille or Dot Decoder, may not be 100% accurate compared to other options. The idea is to make tools available to everyone at no cost, even if they are imperfect. The downside of this is that there is no easy way on this site to generate braille

Let us know if something's not right

Please report anything that's not working for you on the site and we'll fix them as soon as possible. This includes content errors, accessibility issues, improvement or feature ideas, or anything else that will help make Braille Toolbox useful for as many people as possible.
